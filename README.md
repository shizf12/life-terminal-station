# 🕯️ 生命终点站

> "向死而生，方得始终。提前规划，是对生命最大的尊重。"

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://你的用户名.github.io/life-terminal-station)

[![Pure Frontend](https://img.shields.io/badge/Frontend-Vanilla%20JS-orange)](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript)

一个专注于**临终关怀**和**生命规划**的Web应用，帮助用户从容面对生命的终点，提前安排身后事，减轻家人的决策负担，让告别变得温暖而有尊严。

## 📖 项目背景

在中国传统文化中，"死亡"往往是一个避讳的话题。然而，正是因为缺乏提前规划和沟通，许多家庭在面对亲人离世时陷入混乱：财产纠纷、医疗决策分歧、葬礼安排仓促......

**生命终点站**试图改变这一现状：
- 🎯 **消除忌讳**：以理性和温暖的态度面对生命终点
- 📝 **提前规划**：让每个人都能按照自己的意愿安排身后事
- 🤝 **减轻负担**：为家人留下清晰的指引，避免决策困难
- 💝 **传递爱意**：通过告别信等方式，向爱的人表达最后的心意

## ✨ 核心功能

### 1. ⏰ 生命倒计时
可视化展示您的生命剩余时间，以年、月、天、小时、分钟为单位。

**功能特点：**
- 自定义出生日期和预期寿命
- 实时动态倒计时
- 每分钟自动更新
- 紫色调渐变设计，庄重大方

**使用场景：**
> 当用户看到自己的"剩余时间"时，往往会重新审视生活的优先级，珍惜当下，活出更有意义的人生。

### 2. 📜 遗嘱管理
数字化管理您的遗嘱，包括财产分配、受益人指定和特殊嘱托。

**支持内容：**
- ✅ 遗嘱标题和创建时间
- ✅ 详细财产清单（房产、存款、股票、珠宝等）
- ✅ 受益人及分配比例
- ✅ 特殊要求和嘱托事项
- ✅ 多份遗嘱管理（可随时修改更新）

**法律提示：**
> ⚠️ 本应用生成的遗嘱内容可作为参考和备忘，但正式遗嘱建议咨询专业律师，并按要求进行公证和见证。

### 3. 📦 遗物整理
系统化整理个人物品，指定归属和存放位置。

**物品分类：**
- 📄 **证件文件** - 身份证、户口本、房产证、银行卡等
- 💎 **贵重物品** - 珠宝、艺术品、收藏品等
- 🎁 **纪念品** - 照片、信件、旅行纪念品等
- 💻 **电子产品** - 手机、电脑、账号密码等
- 👔 **衣物** - 服装、鞋帽等
- 📦 **其他** - 不属于以上类别的物品

**管理功能：**
- 按类别筛选查看
- 记录存放位置
- 指定接收人
- 添加物品说明和情感价值描述

### 4. ⚰️ 葬礼规划
提前安排葬礼细节，让告别仪式符合个人意愿。

**可配置项：**

| 类别 | 选项 |
|------|------|
| **葬礼形式** | 传统葬礼、火化仪式、生态安葬、海葬 |
| **仪式氛围** | 庄重肃穆、温馨感人、庆祝生命、简单朴素 |
| **音乐选择** | 自定义希望播放的音乐 |
| **主持人** | 指定信任的亲友主持仪式 |
| **出席名单** | 希望到场的重要亲友 |
| **穿着要求** | 对参加者着装的要求 |
| **特殊安排** | 其他个性化需求 |

### 5. ✉️ 告别信
给爱的人留下最后的话，传递未说出口的情感。

**信件类型：**
- 💌 **给家人的信** - 表达爱与感谢
- 👨‍👩‍👧‍👦 **给子女的信** - 人生建议和祝福
- 💕 **给伴侣的信** - 爱的告白和嘱托
- 🤝 **给朋友的信** - 感谢陪伴和回忆

**发送时机：**
- 📤 **立即发送** - 现在就发给对方
- ⏰ **离世后发送** - 由指定人代为发送
- 📅 **指定日期** - 在未来某个特殊日期发送

### 6. 🏥 医疗预嘱（AD - Advance Directive）
提前表达医疗意愿，在无法自主决策时指导医生和家人的选择。

**关键决策：**
- 💔 **急救措施**
  - 是否进行心肺复苏（CPR）
  - 是否使用呼吸机
  - 是否使用鼻胃管/静脉营养
  - 是否进行透析
  
- 🩺 **治疗偏好**
  - 积极治疗：尽一切可能延长生命
  - 舒适护理：以减轻痛苦为主
  - 有限治疗：根据情况灵活决定

- 🛏️ **临终关怀**
  - 希望在哪里离世（医院/家中/临终关怀机构）
  - 疼痛管理方式
  - 是否希望在意识清醒或深度镇静中度过最后时光

- 👤 **医疗代理人**
  - 指定信任的决策代理人
  - 提供代理人联系方式

**意义：**
> 医疗预嘱能避免家人在面对艰难医疗决策时的痛苦挣扎，确保您的意愿得到尊重。

## 🛠️ 技术架构

### 技术栈
- **前端框架**：纯原生 JavaScript (ES6+)
- **样式方案**：CSS3 + CSS Variables
- **存储方案**：LocalStorage（本地存储）
- **部署平台**：GitHub Pages / Vercel / Netlify

### 项目结构
```
life-terminal-station/
├── 📄 index.html          # 主页面（单页应用结构）
├── 🎨 styles.css          # 样式表（响应式设计）
├── ⚙️ app.js              # 应用逻辑（模块化架构）
├── 📖 README.md           # 项目说明
├── 🚀 DEPLOY.md           # 部署指南
└── 📋 LICENSE             # MIT 许可证
```

### 架构特点
- **单页应用（SPA）**：所有功能在一个页面内完成，无需跳转
- **模块化设计**：每个功能模块独立管理（倒计时、遗嘱、遗物等）
- **数据持久化**：使用 LocalStorage 本地存储，刷新不丢失
- **响应式布局**：支持桌面端、平板和手机访问
- **隐私优先**：所有数据保存在用户浏览器，不上传服务器

## 🚀 快速开始

### 在线体验
直接访问已部署的演示站点：
🔗 **https://shizf12.github.io/life-terminal-station**

### 本地运行
1. **克隆仓库**
   ```bash
   git clone https://github.com/你的用户名/life-terminal-station.git
   ```

2. **进入目录**
   ```bash
   cd life-terminal-station
   ```

3. **启动本地服务器**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve
   
   # PHP
   php -S localhost:8000
   ```

4. **浏览器访问**
   打开 http://localhost:8000 即可使用

### 直接打开
由于不使用后端，您也可以直接双击 `index.html` 文件在浏览器中打开（部分浏览器功能可能受限）。

## 📱 使用指南

### 首次使用
1. 设置生命倒计时
   - 点击首页的"设置预期寿命"
   - 输入出生日期和预期寿命
   - 查看您的生命倒计时

2. 探索各项功能
   - 通过顶部导航栏切换不同模块
   - 每个模块都有独立的表单和数据管理

3. 定期更新
   - 遗嘱和医疗预嘱建议定期回顾更新
   - 遗物清单随生活变化持续维护

### 数据管理
**⚠️ 重要提示：**
- 数据存储在浏览器本地（LocalStorage）
- **清除浏览器数据会导致所有记录丢失**
- 建议定期导出重要信息备份
- 如需跨设备同步，请手动复制数据

**导出数据方法：**
1. 在浏览器中按 F12 打开开发者工具
2. 切换到 Application（应用）标签
3. 在左侧选择 LocalStorage → 当前域名
4. 找到 `life_terminal_station` 键值
5. 复制内容保存为文本文件

**导入数据方法：**
反向操作，将备份的 JSON 数据粘贴回 LocalStorage。
